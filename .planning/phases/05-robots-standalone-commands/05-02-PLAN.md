  <action>
    Create `src/commands/robots.ts` exporting `robotsCommand()` that returns a Commander `Command`.

    Follow the exact same pattern as `sitemapCommand()` in `src/commands/sitemap.ts`:

    ```typescript
    import path from 'node:path';
    import { writeFile } from 'node:fs/promises';
    import { Command } from 'commander';
    import { parseRobotsTxt } from '../shared/robots.js';
    import { writeRobotsCsv, ensureDir } from '../io/files.js';
    ```

    **Download subcommand** (`seo robots download <url>`):
    - argument: `<url>` — "URL to fetch robots.txt from (domain or full robots.txt URL)"
    - option: `--output <file>` — "output file for raw robots.txt content", default `'./state/robots.txt'`
    - action handler:
      1. Derive the robots.txt URL: if the input URL does not end with `/robots.txt`, append `/robots.txt` (strip trailing slash first if present, then append `/robots.txt`)
      2. Use `fetch()` (global fetch, NOT the project's fetchPage which uses Playwright) to GET the robots.txt URL. Set User-Agent header to `'seo-audit-cli/2.1'`.
      3. If response is not ok (status >= 400), write error to stderr: `Error: Failed to fetch ${robotsUrl} (HTTP ${response.status})` and set `process.exitCode = 1`, return.
      4. Get response text body.
      5. Ensure the output directory exists via `ensureDir(path.dirname(outputPath))`.
      6. Write the raw text to the output file via `writeFile(outputPath, body, 'utf8')`.
      7. Log to console: `Saved robots.txt to ${outputPath} (${body.length} bytes)`

    **Audit subcommand** (`seo robots audit <url>`):
    - argument: `<url>` — "URL to fetch robots.txt from (domain or full robots.txt URL)"
    - option: `--output <file>` — "CSV output file", default `'./state/robots-audit.csv'`
    - action handler:
      1. Derive the robots.txt URL same as download (append /robots.txt if not present).
      2. Fetch using global `fetch()` with same User-Agent header.
      3. If response is not ok, same error handling as download.
      4. Get response text body.
      5. Parse with `parseRobotsTxt(body)` to get `RobotsDirective[]`.
      6. Ensure output directory exists.
      7. Write CSV via `writeRobotsCsv(outputPath, directives)`.
      8. Log to console: `Robots audit complete. ${directives.length} directives found. Output: ${outputPath}`

    **Important:** Use global `fetch()` (Node 18+ built-in), NOT the Playwright-based `fetchPage()` from `src/shared/http.ts`. Robots.txt is a plain text file — no browser rendering needed. This avoids launching a browser just to download a text file.

    Export: `export function robotsCommand(): Command`
